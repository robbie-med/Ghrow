How this app works

The app is static. It loads a curve catalog from data/catalog.json, fetches the selected CSV file, plots percentile columns from that file, and uses LMS values in the same rows to estimate Z-scores and percentiles for entered measurements.

Saved patient measurements are stored only in the browser. The data files are separate from the JavaScript so the raw source can be inspected, replaced, versioned, and cited.

Clinical caution: this is not a diagnostic device. Use corrected age, curve choice, gestational-age-specific curves, and local policy carefully.
